# StudyStreak Privacy Policy

StudyStreak is a study tracking app with an optional browser extension. This policy explains what data is stored or synced when you use StudyStreak.

## Data Stored Locally

StudyStreak stores your study data on your device, including:

- account profile data for local login
- subjects
- subject websites
- study sessions
- timetable sessions
- achievements
- settings
- focus-quality history

On Windows, installed app data is stored under `%LOCALAPPDATA%\StudyStreak`.

## Data Synced to the StudyStreak Server

If you log in and use cloud sync, StudyStreak may send the following data to the StudyStreak backend:

- username and login credentials for authentication
- encrypted profile data
- subject names
- subject website lists
- timetable sessions
- current streak count for the leaderboard
- focus session minutes
- focus-quality summaries from the browser extension

Focus-quality summaries can include the selected subject, focus score, focused seconds, distracted seconds, idle seconds, top distracted domain, and completion time.

## Browser Extension Data

The StudyStreak Companion browser extension may read the active tab domain while a focus session is running. This is used to decide whether time is focused, distracted, or idle.

The extension does not collect page contents, typed text, browsing history, or passwords. It stores login state, synced subjects, synced websites, timetable reminders, focus settings, and recent focus summaries in browser extension storage.

## Data Sharing

StudyStreak does not sell user data. Cloud data is used to provide account sync, leaderboard streaks, timetable reminders, subject website sync, and focus-quality tracking.

## Offline Use

StudyStreak can be used locally without cloud sync. Signed JSON export/import remains available as an offline fallback for focus-quality summaries.
